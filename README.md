# DragonFix

DragonFix is a Minecraft 1.7.10 Forge mod for small, targeted compatibility and stability fixes.

## Current fixes

- Galaxy Space: prevents `ItemRocket#getRarity` from running client-only rarity logic on the dedicated server.

## Versioning

Build versions are derived from Git by GTNHGradle. Create and push a Git tag to produce a release version:

```bash
git tag 1.0.0
git push origin 1.0.0
```

Local builds from untagged commits will use the Git-derived snapshot version generated by the buildscript.

## Build

```bash
./gradlew build
```

The output jar uses the `DragonFix` archive base name.

## Adding fixes

Add new targeted fixes under `com.dragonfix`, and keep optional mod compile dependencies in `dependencies.gradle`.
